# Веб-сервис спортивного клуба «STALIKA»

Полнофункциональная веб-платформа для автоматизации деятельности клуба настольного тенниса. Система позволяет управлять бронированиями, турнирами и ресурсами клуба в режиме реального времени.

## 🛠 Технологический стек
*   **Backend:** ASP.NET Core MVC (.NET 8)
*   **ORM:** Entity Framework Core (подход Database First/Code First)
*   **База данных:** PostgreSQL
*   **Аутентификация:** Cookie-based Authentication
*   **Отчетность:** ClosedXML (генерация и экспорт данных в формат MS Excel)
*   **Инструменты:** Visual Studio 2022, pgAdmin

## 🚀 Ключевые возможности
*   **Личный кабинет:** Система регистрации и авторизации пользователей.
*   **Бронирование:** Интерактивный сервис бронирования столов, выбора тренера и необходимого инвентаря.
*   **Турниры:** Модуль управления спортивными мероприятиями и сеткой турниров.
*   **Админ-панель:** Полноценный интерфейс для управления пользователями, тренерским составом, инвентарём и расписанием.
*   **Аналитика и отчеты:** Модуль аналитики бронирований с возможностью выгрузки данных в Excel для отчетности.

## 📁 Структура проекта
*   `/Stalika.Web` — Исходный код веб-приложения (Controllers, Views, Models, Data).
*   `/database` — SQL-скрипты и дампы для развертывания базы данных PostgreSQL.

## ⚙️ Инструкция по запуску

### 1. Подготовка базы данных
1. Создайте в PostgreSQL новую базу данных с именем `stalika`.
2. Импортируйте файл дампа из папки `/database/stalika_dump.sql` через Query Tool или функцию Restore в pgAdmin.

### 2. Настройка приложения
1. Откройте решение `Stalika.Web.sln` в Visual Studio 2022.
2. В файле `appsettings.json` обновите строку подключения `DefaultConnection`, указав ваш локальный пароль от PostgreSQL:
   `Host=localhost;Port=5432;Database=stalika;Username=postgres;Password=ВАШ_ПАРОЛЬ`

### 3. Запуск
1. Соберите решение (Build Solution) для восстановления NuGet-пакетов.
2. Запустите проект (F5). Приложение будет доступно в браузере по локальному адресу, указанному в настройках запуска.
